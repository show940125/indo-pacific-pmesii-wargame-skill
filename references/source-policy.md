# Source Policy

Default policy: open/public sources only.

Source tiers:

- `official`: ministries, governments, multilateral institutions
- `public`: established media, public datasets, mainstream analysis
- `mixed`: blended or uncertain provenance
- `social`: social-only signal sources

Rules:

- Social sources are signal inputs, never standalone proof.
- Each key judgment must cite at least two independent source groups.
- Evidence outside mission time window invalidates quality gate unless explicitly annotated.
